# my-valentine

This project is a responsive three‑page birthday surprise website with a Japanese cherry blossom theme. It includes falling petals, gentle animations, and interactive puzzles to guide the user through the celebration.

## Pages

1. **index.html** – Puzzle entry page with a riddle. The user must solve it to proceed. Includes music toggle and unlock animation.
2. **letter.html** – Ancient letter reveal effect. A lamp follows the cursor to "illuminate" handwritten text on parchment.
3. **surprise.html** – Two mystery boxes that open to show surprises with animations, followed by a final birthday message.

## Assets & placeholders

- `style.css` – global styles and responsive design.
- `script.js` – logic for petals, puzzle, lamp, and boxes.
- Placeholder images:
  - `tree-left-placeholder.png`, `tree-right-placeholder.png` – corner sakura trees.
  - `parchment-placeholder.jpg` – background for letter page.
  - `image1-placeholder.jpg`, `image2-placeholder.jpg` – contents of surprise boxes.
  - `puzzle.jpg` – image used for the custom 3x3 sliding puzzle on the entry page (replace with any square image).
  - `music.mp3` – calm background track for page 1.

Replace these placeholders with actual media files to finalize the experience.

## Usage

Open `index.html` in a browser to begin. The site is designed to work on desktop and mobile devices.

JavaScript handles animations and transitions; no build step is required.
